# Codex Email Skill

Static landing page and agent skill for codexemailskill.com. Works with Codex, Hermes, OpenClaw, Claude Code, Cowork, and other skill-aware agents.

Install shortcut:

```bash
npx skills add codexemailskill
npx skills add Sequenzy/codexemailskill --skill codexemailskill
```

Raw skill folder: `skills/codexemailskill`
